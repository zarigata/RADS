## ğŸ”§ Technical Details

### Database API Reference

#### `db.open(path)`
Opens a SQLite database connection.

**Parameters:**
- `path` (string) - Database file path or `:memory:` for in-memory

**Returns:** Boolean (success)

**Example:**
```rads
db.open("myapp.db");
db.open(":memory:");  // In-memory database
```

#### `db.execute(sql)`
Executes INSERT, UPDATE, DELETE, or DDL statements.

**Parameters:**
- `sql` (string) - SQL statement to execute

**Returns:** Integer (affected rows)

**Example:**
```rads
db.execute("CREATE TABLE users (id INTEGER, name TEXT)");
db.execute("INSERT INTO users VALUES (1, 'Alice')");
turbo affected = db.execute("UPDATE users SET name = 'Bob' WHERE id = 1");
```

#### `db.query(sql)`
Executes SELECT queries and displays formatted results.

**Parameters:**
- `sql` (string) - SELECT statement

**Returns:** Integer (row count)

**Example:**
```rads
db.query("SELECT * FROM users");
db.query("SELECT name, age FROM users WHERE age > 18");
```

#### `db.begin()`
Starts a database transaction.

**Returns:** Boolean (success)

#### `db.commit()`
Commits the current transaction.

**Returns:** Boolean (success)

#### `db.rollback()`
Rolls back the current transaction.

**Returns:** Boolean (success)

**Transaction Example:**
```rads
db.begin();
db.execute("INSERT INTO accounts (id, balance) VALUES (1, 1000)");
db.execute("INSERT INTO accounts (id, balance) VALUES (2, 2000)");
db.commit();  // Both inserts succeed or both fail
```

#### `db.close()`
Closes the database connection.

**Returns:** Boolean (success)
