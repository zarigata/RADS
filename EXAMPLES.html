Fibonacci Sequence: 0, 1, 1, 2, 3, 5, 8, 13, 21...

Recursive Approach:

‚îú‚îÄ fib(4)
‚îÇ  ‚îú‚îÄ fib(3)
‚îÇ  ‚îÇ  ‚îú‚îÄ fib(2) ‚Üí 1
‚îÇ  ‚îÇ  ‚îî‚îÄ fib(1) ‚Üí 1
‚îÇ  ‚îî‚îÄ fib(2) ‚Üí 1
‚îî‚îÄ fib(3)

Iterative Approach (Faster!):

üíª Code

// Recursive approach - elegant but slower
blast fibonacci_recursive(i32 n) -> i32 {
    if (n <= 1) {
        return n;
    }
    return fibonacci_recursive(n - 1) + fibonacci_recursive(n - 2);
}

// Iterative approach - faster and more efficient
blast fibonacci_iterative(i32 n) -> i32 {
    if (n <= 1) {
        return n;
    }
    
    turbo i32 prev = 0;
    turbo i32 curr = 1;
    turbo i32 i = 2;
    
    loop (i <= n) {
        turbo i32 next = prev + curr;
        prev = curr;
        curr = next;
        i = i + 1;
    }
    
    return curr;
}

üöÄ Key Concepts

  • Recursion: A function calling itself with smaller inputs
  • Base Case: Condition to stop recursion (n <= 1)
  • Iteration: Using loops for better performance
  • Time Complexity: Recursive O(2^n) vs Iterative O(n)
