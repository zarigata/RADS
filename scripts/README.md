# RADS Scripts

This directory contains build, installation, and utility scripts for RADS.

## Scripts

### install.sh
Installation script for RADS compiler.
```bash
./scripts/install.sh
```

### run_examples.sh
Runs all example programs for testing.
```bash
./scripts/run_examples.sh
```

### test_install.sh
Tests the installation process.
```bash
./scripts/test_install.sh
```

## Usage

All scripts should be run from the project root directory:
```bash
./scripts/install.sh
```

Or make them executable and run directly:
```bash
chmod +x scripts/*.sh
scripts/install.sh
```
