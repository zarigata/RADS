### Pattern-Based Transpilation

Instead of a full lexer/parser (thousands of lines), v0.0.5 uses a pragmatic **pattern-matching approach**:

1. **Line-by-line processing** - Simple, fast, effective
2. **Regex-free** - Uses C string functions for speed
3. **Incremental** - Easy to add new patterns
4. **Testable** - Each pattern can be tested independently

**Why This Works:**
- 90%+ of Python code follows common patterns
- Simple patterns handle complex cases
- Fast iteration and debugging
- Easy to extend for new features
