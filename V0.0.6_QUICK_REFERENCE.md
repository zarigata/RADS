## ğŸŒ WebSocket

### Server Example

```rads
blast main() {
    turbo server = websocket_new(8080);

    websocket_on_message(server, handle_message);
    websocket_on_connect(server, handle_connect);

    echo("WebSocket server listening on port 8080");
    websocket_listen(server);
}

blast handle_message(client, msg) {
    echo("Received: " + msg);
    websocket_send(client, "Echo: " + msg);
}

blast handle_connect(client) {
    echo("New client connected");
    websocket_send(client, "Welcome!");
}
```

### Client Example

```rads
blast main() {
    turbo ws = websocket_connect("ws://localhost:8080");

    websocket_on_message(ws, handle_message);

    websocket_send(ws, "Hello, Server!");
    websocket_wait(ws);  // Keep alive
}

blast handle_message(msg) {
    echo("Server says: " + msg);
}
```

### API Reference

| Function | Description |
|-----------|-------------|
| `websocket_new(port)` | Create new server |
| `websocket_connect(url)` | Connect to server |
| `websocket_send(client, msg)` | Send message |
| `websocket_broadcast(server, msg)` | Broadcast to all |
| `websocket_on_message(target, callback)` | Set message handler |
| `websocket_on_connect(target, callback)` | Set connect handler |
| `websocket_listen(server)` | Start listening |
| `websocket_wait(ws)` | Keep connection alive |

## ğŸ“Š GraphQL

### Server Setup

```rads
blast main() {
    turbo schema = graphql_schema();

    // Define type
    graphql_add_type(schema, "User", [
        "id: ID!",
        "name: String!",
        "email: String"
    ]);

    // Define query
    graphql_add_query(schema, "user(id: ID!): User", get_user);

    // Define mutation
    graphql_add_mutation(schema, "createUser(name: String!): User", create_user);

    // Start server
    graphql_server(8080, schema);
}

blast get_user(args) {
    turbo user = db_query("SELECT * FROM users WHERE id = ?", args.id);
    return user;
}

blast create_user(args) {
    turbo id = db_insert("INSERT INTO users (name) VALUES (?)", [args.name]);
    return { "id": id, "name": args.name };
}
```

### Query Example

```graphql
query GetUser {
  user(id: "123") {
    id
    name
    email
  }
}
```

### Mutation Example

```graphql
mutation CreateUser {
  createUser(name: "Alice") {
    id
    name
  }
}
```

### API Reference

| Function | Description |
|-----------|-------------|
| `graphql_schema()` | Create new schema |
| `graphql_add_type(schema, name, fields)` | Define type |
| `graphql_add_query(schema, name, resolver)` | Add query |
| `graphql_add_mutation(schema, name, resolver)` | Add mutation |
| `graphql_server(port, schema)` | Start server |

## ğŸ¯ LSP Server

### Installation

```bash
cd tools/rads-lsp
cargo build --release
cargo install --path .
```

### VS Code Setup

Install extension and add to `settings.json`:
```json
{
    "rads.languageServerPath": "~/.cargo/bin/rads-lsp"
}
```

### Vim/Neovim Setup

Add to `.vimrc` or `init.lua`:
```vim
autocmd FileType rads lspconfig rads_lsp = {
    \ cmd = { "rads-lsp" },
    \ filetypes = { "rads" },
    \ root_dir_pattern = { ".git/" },
    \}
```

### JetBrains Setup

1. Install "RADS" plugin from Marketplace
2. Configure LSP path in Settings
3. Restart IDE
